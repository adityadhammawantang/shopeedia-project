routing paths for shopeedia e-commerce website


home page
GET     /                   show current catalogs, discounts, etc

authentication
GET     /register           registration page
GET     /login              login page
POST    /register           create new user
POST    /login              login

products & catalog
GET     /catalog            show catalog page
GET     /catalog/:id        get product page with :id also with the comments as well

POST    /catalog/:id/comment    add new comment 

//only admin 
POST    /catalog            add new product 
PUT     /catalog/:id        update product :id information or data
DELETE  /catalog/:id        delete product :id 

shopping cart & checkout
GET     /cart               see shopping cart
GET     /checkout           payment page and confirmation
POST    /checkout           sends confirmation

admin/profile page
GET     /profile/:id        show profile page of :id & with delivery status in one page
GET     /admin              show admin page (dashboard)
